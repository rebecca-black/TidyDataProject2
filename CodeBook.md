CodeBook.md for variable names
Activity_ID – a value from 1 – 6 which maps to the Activity_Name variable as follows:
1	WALKING
2	WALKING_UPSTAIRS
3	WALKING_DOWNSTAIRS
4	SITTING
5	STANDING
6	LAYING

Activity_ID - int, 1 - 6 which stands for one of six activities the subjects perform
Activity_Name -- factor with 6 levels -- textual data added to give the activity ID more meaning for reporting
Subject_ID -- int, 1 - 30, an ID representing the test subjects (30 in total)

The remaining fields are numeric measurements of mean and standard deviation for the X, Y, and Z axis data gathered in the experiment.
These variable names came from the original project, with the code book located at 
http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones
